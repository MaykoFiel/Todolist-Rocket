🚀 TodoList API - Java Spring Boot
Este projeto é uma API REST robusta para o gerenciamento de tarefas (To-Do List), desenvolvida durante o Minicurso de Java da Faculdade de Tecnologia Rocketseat. A aplicação permite o cadastro de usuários e o gerenciamento de tarefas personalizadas com foco em segurança e boas práticas de desenvolvimento back-end. 

🚀 Funcionalidades

- Cadastro de Usuários: Sistema de criação de conta com senhas criptografadas.

- Gerenciamento de Tarefas: Criação, listagem e atualização de tarefas vinculadas a um usuário específico. 

- Filtros de Autenticação: Implementação de segurança para garantir que apenas usuários autorizados acessem suas próprias tarefas. 

- Tratamento de Exceções: Retornos de erro personalizados para uma melhor experiência de integração. 


🛠 Tecnologias e Ferramentas

- Linguagem: Java 17 
- Framework: Spring Boot 3 
- Gerenciador de Dependências: Maven 
- Banco de Dados: H2 Database Engine (em memória para desenvolvimento) 
- Segurança: BCrypt para hashing de senhas e filtros de autenticação personalizada 
- Deploy e Infraestrutura: Docker para containerização da aplicação
